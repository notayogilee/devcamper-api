# DevCapmer API

> Backend API for DevCamper application, which is a bootcamp directory website.

## USage

Rename "config/config.env.env" to "config.env" and update the values/settings to your own

## Install Dependencies

```
npm install
```

## Run App

```

# Run in dev mode
bpn run dev

# Run in prod mode
npm start
```
